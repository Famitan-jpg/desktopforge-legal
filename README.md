# DesktopForge Legal Documents

このリポジトリは、DesktopForge に関する公開用の法務・運用文書をまとめたものです。

## 含まれる文書

- [利用規約 / コンテンツ保護ポリシー](./TERMS.md)
- [プライバシーポリシー](./PRIVACY.md)

## このリポジトリの目的

DesktopForge の配布・運用にあたり、利用条件、サポート方針、知的財産の取り扱い、プライバシー保護の考え方を明示するための公開リポジトリです。

主に以下の内容を対象としています。

- 対応OS、サポート、免責、返金・救済措置に関する基本方針
- クラック、リバースエンジニアリング、AI学習利用、類似ソフト開発、第三者プラグインに関する取り扱い
- 取得する可能性のある情報（クラッシュログ、環境情報など）と、その利用目的

## 注意事項

- これらの文書は、DesktopForge の運営上必要に応じて改定される場合があります。
- 重要な変更がある場合は、可能な限り GitHub 上の更新、関連コミュニティ、配布ページ等を通じて案内します。
- 各文書の最新版は、このリポジトリ上のファイルを基準とします。
